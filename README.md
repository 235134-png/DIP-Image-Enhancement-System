# DIP-Image-Enhancement-System

## Project Description
This project is an integrated **Smart Image Enhancement & Analysis System** developed for **Lab 06: Digital Image Processing**. It demonstrates a complete pipeline from image acquisition to advanced enhancement using MATLAB.

## Features
* **Lab 01:** Image Acquisition & Metadata Reporting.
* **Lab 02:** Sampling (Down/Up) and Bit-Depth Quantization (4-bit).
* **Lab 03:** Geometric Transformations (45° Rotation and Translation).
* **Lab 04:** Intensity Transformations (Logarithmic & Gamma Correction).
* **Lab 05:** Histogram Processing (Global Histogram Equalization).
